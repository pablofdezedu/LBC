TICO.1.C.3. Hojas de cálculo

Comenzamos

Comenzamos el trimestre con el uso y manejo de la herramienta Hojas de cálculo. Veremos de forma gradual esta potente aplicación realizando otra situación de aprendizaje, utilizando el modelo de empresa que seleccionó en el anterior trimestre, de manera que podamos crear unos nuevos documentos para la misma.

Herramienta de trabajo:

Puedes elegir a voluntad cualquiera de las dos aplicaciones disponibles

  • Google Hojas de cálculo (preferible por tener acceso a los documentos en línea)
  • Libreoffice Calc  (Software libre, enlace disponible en la página de inicio del curso)

Si dispones de licencia para otro programa diferente, puedes utilizarla.

La mayoría de las opciones funciones o fórmulas que vamos a utilizar están presentes en todos las aplicaciones citadas.
