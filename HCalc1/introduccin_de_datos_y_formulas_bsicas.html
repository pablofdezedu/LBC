Introducción de datos y formulas básicas

Primera parte

En nuestro ejemplo, que podrá ver en un vídeo mas adelante, realizaremos una hoja de cálculo para contabilizar las canastas de un partido de baloncesto.

  • Damos formato a la hoja con encabezados adecuados para que se sepa que información estamos colocando.
Imagen propia. jjpriets. Encabezados (CC BY-SA)
  • Rellenar la matriz de datos con "Canastas o puntos" de los jugadores
Imagen propia. jjpriets. Datos (CC BY-SA)
  • Calcular el total de puntos, recuerda realizar BIEN la primera fórmula para poder EXTENDER. Puedes utilizar los siguientes métodos
    • = B3+C3+D3+E3  y extender
    • =Suma ( RANGO DE CELDAS)
Imagen propia. jjpriets. Total (CC BY-SA)
  • Calcular la Media aritmética dividiendo entre el número de valores sumados. 
    • = casilla de suma / 4   ->    = F3 / 4
Imagen propia. jjpriets. Media (CC BY-SA)
  • Calcular la media utilizando la función PROMEDIO del rango de valores deseados.
    • =PROMEDIO (rango de celdas)
Imagen propia. jjpriets. Promedio (CC BY-SA)
  • Resultado activando Mostrar fórmulas para ver el contenido de las celdas con fórmulas
Imagen propia. jjpriets. Fórmulas (CC BY-SA)
