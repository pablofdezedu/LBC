Gráficos Ejemplos

Los siguientes ejemplo con gráficos muestran diferentes tipos de representación de la información.

Columnas 

Sobre una tabla de datos, realizar la representación gráfica de unas columnas concretas.

Gráfico deseado:

Jjpriets. Gráfico a mostrar (CC BY-NC-SA)

El siguiente ejemplo muestra unos contenidos con datos no relevantes para el gráfico que queremos realizar.

Jjpriets. Datos de origen (CC BY-NC-SA)

Seguir la siguiente operativa para la creación del gráfico:

  • Selección de datos:
    • Incluir los rótulos horizontales y verticales necesarios para que el gráfico pueda utilizarlos como información
    • Seleccionar los datos con el ratón
    • Utilizando la tecla CONTROL podemos realizar selección de regiones que están separadas.
Jjpriets. Selección parcial de información (CC BY-NC-SA)
  • Acceder al menú Insertar -> Gráfico
    • Seleccionar el tipo de gráfico..
    • Cambiar a la pestaña "personalizar" para ajustar:
      • Títulos de gráfico
      • Posición de leyenda
      • Formato de eje horizontal
      • Formato del eje vertical
      • Líneas y cuadrícula

Sobre el mismo gráfico se pueden realizar algunos cambios de datos haciendo clic sobre el elemento deseado

Para imprimir solamente el gráfico seleccionar la región (CELDAS) sobre las que está el gráfico y acceder a la impresora que está en la barra de herramientas. En la primera de las opciones "Imprimir" seleccionar "Celdas seleccionadas" para visualizar solamente esa región.

Circular

En esta ocasión vamos a representar el siguiente gráfico

Jjpriets. Gráfico de sectores (CC BY-NC-SA)

Dados los siguientes datos procedemos a seleccionar toda la tabla.

  • Selección de datos
    • Con la ayuda del ratón seleccionar todos los datos, los rótulos serán necesarios para que el gráfico los utilice como datos de etiquetas.

 

Jjpriets. Selección de datos (CC BY-NC-SA)
  • Acceder al menú Insertar-> Gráfico
    • Seleccionar Circular 3D
    • Acceder al menú "Personalizar"
      • Opción "Gráfico circular":
        • Etiqueta de sector seleccionar Valor y porcentaje (para mostrar los datos que representan cada sector). Puedes reducir la fuente para que quepa dentro de cada porción.
      • Leyenda: Cambiar a "Abajo"

Se puede cambiar el título del gráfico dentro de la opción que ofrece el menú anterior o haciendo doble clic
